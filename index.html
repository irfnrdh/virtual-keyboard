Papan Ketik Cadangan

Keyboard virtual untuk keyboard yang rusak

Klik tombol-tombol di bawah seperti keyboard biasa. Hasil ketikan akan muncul di kotak teks — tinggal salin (copy) lalu tempel (paste) ke aplikasi lain. Tombol fisik yang masih berfungsi tetap bisa dipakai langsung di kotak teks.

Hasil Ketikan 0 karakter
⇧ Shift: Off ⇪ Caps Lock: Off Aktifkan Shift / Caps Lock di sini, lalu ketik pakai tombol fisik atau klik tombol virtual di bawah. Aktifkan Auto-Salin agar setiap tombol yang diklik langsung tersalin ke clipboard.
Terdeteksi berfungsi Sedang ditekan ⌨️ Caps Lock fisik: belum dites 🧪 Mode Tes Keyboard: tekan tombol fisik di keyboardmu satu per satu (termasuk Shift & Caps Lock). Tombol yang sesuai di bawah akan menyala hijau jika sinyalnya terdeteksi oleh browser. Kalau tidak menyala sama sekali saat ditekan berkali-kali, kemungkinan besar tombol fisik itu yang rusak — pakai versi virtualnya. Pill Shift/Caps Lock di atas adalah tombol virtual yang kamu kontrol sendiri, terpisah dari status fisik ini.
Tombol Karakter
Dibuat untuk membantu mengetik saat keyboard fisik rusak atau ada tombol yang tidak berfungsi.
Catatan: tombol Fn tidak bisa dideteksi oleh halaman web mana pun (bekerja di level chip keyboard, bukan dikirim ke browser) — gunakan tombol Home / End / Delete / ←→ di keyboard virtual sebagai gantinya.
